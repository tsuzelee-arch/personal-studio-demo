# 提示詞庫與熔爐優化計劃

這份計劃將針對您提出的六項需求進行系統性的優化與實作：

## 1. 介面與文案優化
- **更名熔爐標題**：將「提示詞熔爐—拖拽合成」簡化更名為「提示詞熔爐」。
- **新增拖曳區說明**：在提示詞熔爐下方的空白文字框區域，加入「可將提示詞模塊拖拽于此處合成。」的提示說明，引導使用者操作。

## 2. 熔爐 AI 轉換功能升級
- **自訂轉換模型**：在「轉自然語言」與「轉 SD Tags」按鈕旁，新增**模型選擇器**（與圖像解構的模型列表同步），讓您可以自由選擇要用哪一個 AI 模型來執行轉換。
- **目標語言選項**：新增**目標語言選擇器**，在點擊「轉自然語言」或「轉 SD Tags」時，AI 將依據您指定的語言輸出結果。

## 3. 熔爐文字框操作優化 (Ctrl+Z 支援)
- 目前拖曳提示詞到熔爐時，是直接覆寫系統底層的值，這會破壞瀏覽器原生的「復原 (Undo)」歷史紀錄。
- **優化實作**：將拖曳寫入的行為改寫為透過 `document.execCommand('insertText')`（或利用 Rich Editor 的 DOM 插入行為並觸發原生 input 事件），這樣就能完美支援 `Ctrl+Z` 復原剛拖曳進去的提示詞！

## 4. 提示詞語言屬性與同步
- **新增語言屬性**：在「新增 / 編輯提示詞」的彈出視窗（Modal）中，加入「語言 (Language)」選項。
- **從圖像解構儲存時同步**：當您在「圖像解構」面板中點擊「儲存至提示詞庫」時，系統會自動捕捉當下解構所設定的「輸出語言」，並在儲存卡片時自動帶入該語言屬性。

## 5. 新增提示詞的分類自動連動
- **分類預選邏輯**：當您在左側側邊欄點選了特定的分類（例如「風格」或「光影」）後，若此時點擊「新增提示詞」，彈出視窗內的分類下拉選單將會**自動預設為您當前選取的分類**（若選取的是「全部」，則保持預設值），省去重複選擇的麻煩。
