          啟用本機 Gemini Nano 說明

          目前偵測到您的瀏覽器尚未啟用或未下載本機模型。請依照以下步驟啟用 Chrome 內置的 Gemini Nano 模型：

          1. 打開 Chrome 新分頁，前往 chrome://flags/#optimization-guide-on-device-model，將其設為 Enabled BypassPrefRequirement。
          2. 前往 chrome://flags/#prompt-api-for-gemini-nano，將其設為 Enabled。
          3. 點擊 Chrome 右下角的 Relaunch 按鈕重新啟動瀏覽器。
          4. 重新啟動後，前往 chrome://components，找到 Optimization Guide On Device Model，點擊 Check for update（檢查更新）以確保模型下載完成（檔案較大，下載可能需要數分鐘）。
          提示： 啟用後，即使在斷網或沒有設定任何 API 金鑰的狀態下，您也可以使用「提示詞熔爐」中的「轉自然語言」與「轉 SD Tags」功能。
