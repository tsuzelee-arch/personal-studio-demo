# 客制化 Studio 操作介面教程 (STUDIO_USER_GUIDE.md)

歡迎使用 **客制化 Studio**！本手冊旨在引導您掌握客制化 Studio 介面的各項功能與操作方式，讓您能順暢地進行 Creative AI 創作與工作流設計。

---

## 🧭 介面佈局概覽

啟動系統後，主介面分為三大區塊：
1.  **左側導航列 (Sidebar)**：切換不同的工作室面板（簡易工具流、圖像解構、提示詞庫與熔爐、資產庫、API 設定）與切換色彩模式。
2.  **頂部工具列 (Topbar)**：顯示目前面板標題，並提供全域 IDE Agent 助手的啟動按鈕。
3.  **主內容區 (Content Area)**：展示當前選定面板的詳細工作界面與操作畫布。

---

## 🎨 簡易工具流 (Simple Workflow) 畫布操作指南

簡易工具流面板提供直覺的卡片式畫布，用於編排圖像生成任務。

### 1. 建立與編輯節點
*   **文生圖 (T2I) 節點**：點擊畫布上方「➕ 文生圖」建立。可在卡片中輸入提示詞，進行純文字生成圖像。
*   **圖生圖 (I2I) 節點**：點擊畫布上方「➕ 圖生圖」建立。除了提示詞，還可上傳一張或多張參考圖片，控制生成圖像的形體與細節。
*   **新增群組**：點擊「🗂 新增群組」建立一個可拖曳的半透明背景框，將相關卡片放進群組中以便一起移動與分類。

### 2. 畫布導航與檢視（滑鼠與鍵盤操作）
*   **放大／縮小**：在畫布任意處滾動**滑鼠滾輪**以調整畫布縮放比例。
*   **平移畫布**：按住**滑鼠中鍵**（滾輪鍵）並拖曳，或按住**鍵盤空白鍵 (Space)** 同時按住滑鼠左鍵拖曳。

### 3. 參考圖片快捷操作 (Hotkeys & Clipboard)
在「圖生圖 (I2I)」節點中，您可以使用高度流暢的快捷鍵管理參考圖片：
*   **選取圖片**：點擊 I2I 節點中的任意一張參考縮圖，該圖片會呈現**藍色高亮外框**表示已被選取。
*   **複製圖片 (`Ctrl + C`)**：選取圖片後，按 `Ctrl + C`，或將滑鼠懸停於縮圖上並點擊出現的「複製 (❐)」按鈕。圖片會寫入系統剪貼簿。
*   **智慧貼上 (`Ctrl + V`)**：
    *   **無選取節點時**：在畫布空白處按 `Ctrl + V`，系統會自動在**目前螢幕視野正中心**建立一個全新的 I2I 節點，並自動將剪貼簿圖片填入參考圖。
    *   **已選取 I2I 節點時**：在該選取狀態下按 `Ctrl + V`，圖片會直接被附加至該節點的參考圖片清單中。
*   **單獨刪除圖片 (`Delete` / `Backspace`)**：點選特定縮圖高亮後，按 `Delete` 或 `Backspace` 即可**僅刪除該張圖片**，而不會誤刪整張節點卡片。

## 🔍 圖像解構 (Image Decode) 操作指南

當您擁有一張精美圖片並想探究其提示詞、色板與細節時，請使用圖像解構面板。

### 1. 上傳並分析
*   點擊解構區塊的虛線框上傳圖片，或直接將本地圖片檔案拖曳至框內。
*   點擊「開始解構」，系統將運用 AI Vision 模型分析圖片。

### 2. 解析解構結果與收納 (Vault)
*   分析完成後，介面會呈現結構化的圖像資訊：
    *   **視覺主體與風格描述**（AI 提示詞）。
    *   **色板 (Color Palette)**：自動提取畫面主要的 Hex 色碼，並生成色塊。
    *   **細節清單**：光影細節、環境中景與背景描述、畫面材質及相機鏡頭參數。
*   **收納功能**：懸停在任意提取出的細節文字或色板上，會出現「**➕ 收納**」按鈕，點擊即可**一鍵將其儲存至提示詞庫**中，免去手動複製貼上的繁瑣步驟。

## 📚 提示詞庫與熔爐 (Prompts) 操作指南

管理您珍藏的提示詞，並使用 AI 熔爐對提示詞進行再創作與翻譯。

### 1. 分類管理
*   **切換分類**：點擊左側分類側邊欄的分類項目，主畫面即會篩選出該分類下的所有提示詞卡片。
*   **新增分類**：點擊側邊欄底部的「**+ 新增分類**」，輸入名稱即可新增。
*   **刪除分類**：將滑鼠懸停在您建立的自訂分類上，點擊出現的 `✕` 按鈕，系統會彈出確認視窗。確認刪除後，該分類下的所有提示詞將被自動安全地移至「其他元素 (Other)」中，不會遺失提示詞。

### 2. 提示詞熔爐 (Prompt Forge)（右側面板）
當您點擊或新增提示詞時，右側的熔爐會開啟：
*   **AI 重寫 (Rewrite to Prompt)**：輸入一段凌亂的靈感或簡單文字，點擊「AI 重寫」，AI 會將其潤飾為高畫質、細節豐富的繪圖提示詞。
*   **轉為 SD 標籤 (Convert to SD Tags)**：將提示詞內容一鍵轉換為 Stable Diffusion 常用的逗號分割英文標籤（Tags）格式。
*   **AI 翻譯**：可將提示詞在繁體中文、簡體中文與英文之間快速翻譯。

## 📦 資產庫 (Assets) 與 API 設定 (Settings)

*   **資產庫**：展示您在工作流中生成的所有創意結果，支援圖片瀏覽、重新命名及下載儲存。
*   **API 設定**：
    *   **AI 金鑰**：請在對應欄位輸入您的 OpenAI API Key、Gemini API Key。
    *   **網絡設定**：若您在特定網絡環境下，可在此設定 Proxy 伺服器代理網址。
    *   **雲端備份**：填入您的 Google Drive Client ID 以啟用 GIS 授權，將個人作品與工作流直接儲存與備份至您的雲端硬碟。

## ✨ 全域 IDE Agent 助手操作

*   在介面任何地方，點擊右上角的 `✨` 按鈕，可於螢幕右側滑出一個獨立的 Chat 視窗。
*   您可以直接在此對話框中向 AI 助手輸入開發指令、詢問繪圖建議、或是直接傳送圖片要求進行圖像修改與生成。點擊 `✕` 按鈕可關閉面板，您的對話歷史會被安全保留。
